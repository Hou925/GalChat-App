# GalChat App

一款基于 Vue 构建的 AI 智能聊天应用。在 GalChat 中，你不仅可以像使用常规线上通讯软件一样与真实世界的好友畅聊，还能自由创建并定制专属的虚拟 AI 角色，体验前所未有的个性化交互。

## 核心功能

*   **真实好友通讯**：提供稳定流畅的基础聊天功能，随时随地与现实中的好友保持联系。
*   **定制虚拟角色**：自由创建你的专属 AI 角色，并深度配置TA的性格特征、说话口吻等属性。
*   **情感与日程感知**：虚拟角色能够智能读取你所设定的“当前心情”和“个人日程”，从而给出更加贴心、个性化的回复内容。
*   **现代交互体验**：核心前端采用 Vue.js 打造，提供丝滑、响应迅速的用户界面。

## 快速开始

### 环境要求

*   Node.js (建议 v14 及以上版本)
*   npm 或 yarn

### 安装与运行

安装项目依赖：
   ```bash
   cd GalChat-App
   npm install
   ```

启动本地开发服务器：
   ```bash
   npm run serve
   ```
