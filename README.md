# GalChat App

An AI-powered smart chat application built with Vue. GalChat allows you to seamlessly chat with both your real-world friends and highly customizable AI virtual characters, bringing a new level of personalized interaction to your daily communication.

## Features

*   **Real-World Messaging**: Connect and chat with your real-life friends just like any standard online messaging application.
*   **Custom AI Characters**: Freely create virtual AI companions and define their unique personality traits.
*   **Context-Aware AI**: The AI characters can read your configured "current mood" and "daily schedule" to generate highly personalized and context-aware responses.
*   **Modern UI/UX**: Built heavily with Vue.js to provide a smooth, responsive, and reactive user interface.

## Getting Started

### Prerequisites

*   Node.js (v14 or above recommended)
*   npm or yarn

### Installation

Install dependencies:
   ```bash
   cd GalChat-App
   npm install
   ```

Run the development server:
   ```bash
   npm run serve
   ```
